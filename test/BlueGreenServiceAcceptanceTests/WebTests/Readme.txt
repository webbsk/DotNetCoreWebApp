The webtest in this folder is used to create a load test definition in VS Online. Ex: 
https://microsoft.visualstudio.com/DefaultCollection/Universal%20Store/_apps/hub/ms.vss-cloudloadtest-web.hub-loadtest-test?_a=requests&name=ReferenceServiceLoadTests&definitionId=683faf78-5888-4e14-ae9e-cee38b09150c
All the parameters of the load test is defined in the settings for load test within VS load test plan. Web Test merely acts as test container for load test.
